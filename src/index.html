HTML Introduction

A Simple HTML

  1. <!DOCTYPE html>
    • Declaration repsents the document type, and helps browsers to display web pages correctly
  2. <html>
    • the root element and it defines the whole HTML document
  3. <body>
    • document's body
  4. Elements
    • everything from the start tag to the end tag
    • <tagname>Content...</tagname>
  5. Attributes
    • provide additional information about HTML elements.
    • href in <a> tag
    • style: used to add styles to an element, such as color, font, size, and more
    • lang in <html>: declare the language of the Web page. This is meant to assist search engine and browsers
    • title: defines some extra information about an element
  6. Headings
    • <h1>
    • <h2>
    • <h3>
    • <h4>
    • <h5>
    • <h6>
  7. Paragraphs
    • <p>
    • <hr>: defiens a thematic break in an HTML page (horizontal rule)
    • <br>: defines a line break
    • <pre>: displayed in a fixed-width font (usually Courier), and it preserves both spaces and line breaks
  8. Links
    • <a href="https://www.link.com/">This is link</p>
  9. Images
    • <img src="https://www.link.com/" alt="alternative text" width="104" height="142" />
  10. Formatting
    • <b>: Bold
    • <strong>: Important text
    • <i>: Italic
    • <em>: Emphasized
    • <mark>: Marked text
    • <small>: Smaller text
    • <del>: Deleted text
    • <ins>: Inserted text
    • <sub>: Subscripttext
    • <sup>: Superscripttext
  11. Quotations
    • <blockquote>: defines a section that is quoted from another source

      Here is a quote from WWF's website:

      For 60 years, WWF...
    • <q>: defines a short quotation

      WWF's goal is to: Build a future where people live in harmony with nature.

    • <abbr>: defines an abbreviation or an acronym like HTML, CSS, ...

      The WHO was founded in 1948.

    • <address>: defines the contact information for the author/owner of a document or an article
      example:
      Written by John Doe.
      Visit us at:
      Example.com
      Box 564, Disneyland
      USA
    • <cite>: defines the title of a creative work

      The Scream by Edvard Munch. Painted in 1893.

    • <bdo>: used to override the current text direction This text will be written from right to left
